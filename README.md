# Country fetch API (GeoGet)

```html
<p>
 -Get information about countries.
 -Just clone the repository, download dependencies and run the code.

## Personal Spring Boot project calling external API using WebClient

### Rest Countries API v3 support

https://restcountries.com/

### Rest Countries API official documentation

https://restcountries.com/#endpoints

<section>

 .../api/v1/countries/region/...

```
